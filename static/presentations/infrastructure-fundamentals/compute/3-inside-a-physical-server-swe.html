Server Hårdvaruarkitektur

Vad Gör en Server Annorlunda?

Servrar hanterar produktionsarbetsbelastningar och fungerar kontinuerligt, ofta i flera år utan avstängning

Hårdvara designad för tillförlitlighet, servicebarhet och prestanda under varaktig belastning

Samma grundläggande komponenter som stationära datorer, men optimerade för serverarbetsbelastningar

Central Processing Unit (CPU)

CPU:n exekverar instruktioner som utgör alla serveroperationer

Flera kärnor (8, 16, 32+) möjliggör parallell exekvering av uppgifter

Intel Xeon och AMD EPYC processorer dominerar servermarknaden

Dual-socket konfigurationer stödjer två fysiska CPU:er, vilket fördubblar kärnor och minnesbandbredd

Stöd för ECC memory och avancerade funktioner för tillförlitlighet

Random Access Memory (RAM)

RAM tillhandahåller temporär lagring för data som CPU:n aktivt bearbetar

ECC (Error-Correcting Code) memory upptäcker och korrigerar single-bit fel automatiskt

Förhindrar krascher och datakorruption från kosmisk strålning eller elektriska störningar

Server RAM-kapacitet: 16 GB (små arbetsbelastningar) till flera terabyte (stora databaser)

Minnesintensiva applikationer som in-memory databaser kräver avsevärda mängder RAM

Storage Drives

HDDs (Hard Disk Drives): Snurrande magnetiska plattor, hög kapacitet, lägre kostnad, mekanisk rörelse begränsar hastigheten

SSDs (Solid-State Drives): Flash memory, mycket snabbare, inga rörliga delar, högre kostnad per gigabyte

RAID (Redundant Array of Independent Disks): Kombinerar flera diskar för redundans eller prestanda

RAID 1: Speglar data över två diskar för redundans

RAID 10: Kombinerar spegling med striping för både redundans och prestanda

Infrastrukturkomponenter

Motherboard: Ansluter alla komponenter, stödjer flera CPU:er och stora mängder RAM, inkluderar BMC för fjärrhantering

Power Supply Unit (PSU): Konverterar AC till DC-ström, redundanta PSU:er för kontinuerlig drift, hot-swappable design

Cooling Systems: Fläktar och kylflänsar upprätthåller säkra driftstemperaturer, justerar hastighet baserat på sensorer

Nätverks- och Expansionsmöjligheter

Network Interface Cards (NICs): Stödjer 1 Gbps, 10 Gbps, 25 Gbps eller snabbare hastigheter

Flera NICs för produktionstrafik, hanteringsåtkomst och lagringsnätverk

Hardware offload-funktioner frigör CPU-cykler för applikationsarbete

PCIe Expansion Slots: Lägg till specialiserad hårdvara som GPUs, storage controllers eller säkerhetsmoduler

Olika slot-storlekar (x4, x8, x16) erbjuder varierande bandbreddskapacitet

Design för Kontinuerlig Drift

Hot-swappable komponenter: Byt ut felaktiga diskar eller PSU:er utan avstängning

Rack-mountable chassis: Standard 19-tums racks, mäts i rack units (U), möjliggör högdensitets-utplaceringar

Remote management interfaces: IPMI, iLO (HP), iDRAC (Dell) erbjuder out-of-band åtkomst

Slå på/av servrar, få åtkomst till konsolen, övervaka hårdvaruhälsa på distans

Nödvändigt för att hantera servrar i avlägsna datacenter

Fysiska servrar innehåller samma grundläggande komponenter som stationära datorer, men i konfigurationer designade för kontinuerlig drift, tillförlitlighet och servicebarhet
