Infrastructure Fundamentals

Server Hardware Architecture

What Makes a Server Different?

Servers handle production workloads and operate continuously, often for years without shutdown

Hardware designed for reliability, serviceability, and performance under sustained load

Same fundamental components as desktop computers, but optimized for server workloads

Central Processing Unit (CPU)

The CPU executes instructions that constitute all server operations

Multiple cores (8, 16, 32+) allow parallel execution of tasks

Intel Xeon and AMD EPYC processors dominate the server market

Dual-socket configurations support two physical CPUs, doubling cores and memory bandwidth

Support for ECC memory and advanced features for reliability

Random Access Memory (RAM)

RAM provides temporary storage for data the CPU is actively processing

ECC (Error-Correcting Code) memory detects and corrects single-bit errors automatically

Prevents crashes and data corruption from cosmic rays or electrical interference

Server RAM capacity: 16 GB (small workloads) to multiple terabytes (large databases)

Memory-intensive applications like in-memory databases require substantial RAM

Storage Drives

HDDs (Hard Disk Drives): Spinning magnetic platters, high capacity, lower cost, mechanical movement limits speed

SSDs (Solid-State Drives): Flash memory, much faster, no moving parts, higher cost per gigabyte

RAID (Redundant Array of Independent Disks): Combines multiple drives for redundancy or performance

RAID 1: Mirrors data across two drives for redundancy

RAID 10: Combines mirroring with striping for both redundancy and performance

Infrastructure Components

Motherboard: Connects all components, supports multiple CPUs and large amounts of RAM, includes BMC for remote management

Power Supply Unit (PSU): Converts AC to DC power, redundant PSUs for continuous operation, hot-swappable designs

Cooling Systems: Fans and heat sinks maintain safe operating temperatures, adjusts speed based on sensors

Network & Expansion Capabilities

Network Interface Cards (NICs): Support 1 Gbps, 10 Gbps, 25 Gbps or faster speeds

Multiple NICs for production traffic, management access, and storage networks

Hardware offload capabilities free CPU cycles for application work

PCIe Expansion Slots: Add specialized hardware like GPUs, storage controllers, or security modules

Different slot sizes (x4, x8, x16) provide varying bandwidth capabilities

Design for Continuous Operation

Hot-swappable components: Replace failed drives or PSUs without shutdown

Rack-mountable chassis: Standard 19-inch racks, measured in rack units (U), enables high-density deployments

Remote management interfaces: IPMI, iLO (HP), iDRAC (Dell) provide out-of-band access

Power servers on/off, access console, monitor hardware health remotely

Essential for managing servers in remote data centers

Physical servers contain the same fundamental components as desktop computers, but in configurations designed for continuous operation, reliability, and serviceability
