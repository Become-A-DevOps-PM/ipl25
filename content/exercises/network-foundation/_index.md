# Network Foundation

Learn the fundamentals of Azure networking through progressive exercises, from visual portal interface to command-line automation.

## Learning Path

This track guides you through virtual network provisioning and security configuration using two complementary approaches:

1. **Portal Interface** - Graphical approach for understanding Azure network resources and security groups visually
2. **Command Line Interface** - Scriptable approach for network automation and repeatability
