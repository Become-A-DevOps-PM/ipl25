# Server Foundation

Learn the fundamentals of Azure infrastructure through progressive exercises, from visual portal interface to command-line automation.

## Learning Path

This track guides you through server provisioning and configuration using two complementary approaches:

1. **Portal Interface** - Graphical approach for understanding Azure resource management visually
2. **Command Line Interface** - Scriptable approach for infrastructure automation and repeatability
